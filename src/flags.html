In this section, we will see new instructions at the same time as the flags, to help understand how instructions work with flags.

The f register

f being an 8-bit register, it has 8 bits. Bits 0-3 are always zero. Bits 4-7 are called C, H, N, and Z, in that order. Note that, to avoid confusing register c and flag c, flags sometimes get an "f" appended to them (eg. CF, ZF).

As always, each of these letters stands for something:

c
Carry
h
Half-carry
n
Add/Subtract (ok, this one doesn't make sense)
z
Zero
